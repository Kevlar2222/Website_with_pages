Important HTML tags

This page is designed as a quick reference for some of the common tags in html. Home page.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header header element useful for CSS
div division divides data for CSS
ol ordered list create ordered list
ul unordered list create unordered list
li list element numbers or bullets for list item

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people<strong> at least a billion people
a anchor make hyperlink <a href="index.html">Home Page</a> Home page
img image include an image <img src="images/kitten.jpg" alt="An adorable kitten">
code code display as code <code>This is code<code> This is code
span span separates inline data for CSS <span>Call me Ishmael.</span> Call me Ishmael.

Document tags

Tag Name Purpose
html html Begins and ends the html document
head head Contains title and data
body body Defines main visible content
title title Declares title
meta metadata Data about data
a anchor Defines a link
